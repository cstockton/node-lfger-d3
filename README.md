lfger-d3 -- Javascript library for diablo III
==============================

## SYNOPSIS

This is a library to be used via a web browser or nodejs app.

### Install

To install this into your current directory use npm

    cd /your/project/directory/
    npm install lfger-d3
    cd lfger-d3
    make test
